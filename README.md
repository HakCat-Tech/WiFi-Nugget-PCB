# WiFi Nugget PCB
This repository contains the PCB design files for the ESP8266 based WiFi Nugget.

### Change Log + Specs
**Beta-v1.0:**  
This design is a proof of concept for a slimmer, more easily manfacturable design based off surface mount components.   
- 0603 SMD components
- ESP12F based module
- SH1106 OLED Screen
- Minimal PCB art
